# Admin Frontend 🏀

React admin panel for NBA 2K Legacy Vault.

## Features
- Login with password
- 15 admin tabs (Games, Clips, Deploy, Monitor, etc.)
- Black/Red/White theme matching the main site

## Setup

```bash
npm install
npm run dev
```

## Environment
- `VITE_API_URL` - Backend API URL (default: http://localhost:8001)

## Deploy to Vercel
1. Connect repo to Vercel
2. Add `VITE_API_URL` environment variable
3. Deploy!

## Tabs
1. Acceleration - AI agent chat
2. Deploy - Live deployment
3. Monitor - System health
4. Neplit - Build tools
5. Games - Game management
6. Clips - Video clips
7. Mockups - UI mockups
8. Proof - Evidence images
9. Community Wall - Social posts
10. Live Feed - Social feed
11. Submissions - User submissions
12. Content - Site content
13. Comments - Comments
14. Emails - Subscribers
15. Petition - Signatures
